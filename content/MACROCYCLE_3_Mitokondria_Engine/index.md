# MACROCYCLE 3 — MITOKONDRIA ENGINE

**Window:** 2028Q1 – 2028Q4 (~52 minggu)
**Trajectory:** BB 63kg → 66kg | BF 11% → 10% | Aerobic engine + LBM build
**Mesocycle Order:** Mito Base Build → Monster Mito Build → Hybrid Strength-Endurance → Performance Peak

---

Macrocycle 3 = aerobic engine maximization. 4 mesocycle volume scaling 8-11 → 11-13 → 13-14.5 → 14.5-16 jam/minggu. Polarized Z2/Z4 distribution, brick session intro. Peptide mito layer (MOTS-c + SS-31 + NAD+ IV) sebagai foundation, BPC-157+TB-500 cycle untuk tendon defense.

## SCORING FRAMEWORK

Peptide pilih **Endurance score (Mito component)** dominant: MOTS-c (mito-derived peptide), SS-31 (cardiolipin-targeting tetrapeptide, FDA Forzinity 2025), NAD+ IV (sirtuin substrate). Recovery layer: BPC-157, TB-500 untuk tendon load tolerance.

Vitamin: Creatine 5g daily, CoQ10 Ubiquinol 100mg (mito support), Iron baseline (foot-strike volume tinggi).

## MESOCYCLE 1: MITO BASE BUILD (2028Q1)

**Context:** Mito stack onboarding. Z2 8-11h/wk + brick 1×/wk. Foundation untuk monster build Q2.

**Target Rationale:** BB 63→64 maintain mild surplus, BF 11→11% stable. Volume increment +2-3h/wk dari MC2. Aerobic markers: VO2max + lactate threshold baseline test (re-measure W12).

**Peptide Rationale:**
- **MOTS-c** ⭐ DOMINANT — 5mg 3×/wk Sen Rab Sab pre-cardio (AMPK activation)
- **SS-31** ⭐ — 10mg cycle 5wk on / 18wk off (cardiolipin mito ATP)
- **NAD+ IV** ⭐ — 100mg 3×/wk Sen Rab Jum (ATP + sirtuin substrate)
- **BPC-157 + TB-500** continue cycle (tendon defense volume)

**Cardio Rationale:** Z2 8-11h/wk polarized 80/20. Walking out, running in (gradual). Brick 1× Sabtu (bike → run 45+30 min). Swim 2×/wk technique.

**Gym Rationale:** 3×/wk compound + unilateral. Volume maintain MC2 baseline. RPE 7.

**Vitamin Rationale:**
- Tier S continue
- **Creatine 5g** continue
- **CoQ10 Ubiquinol 100mg** ⭐ intro (mito support)
- Iron monitor quarterly (volume scaling)

**Transition → Mesocycle 2 (2028Q2 Monster Mito):** End of W12. Aerobic markers improvement, no overuse injury. Volume scale up 11-13h/wk.

## MESOCYCLE 2: MONSTER MITO BUILD (2028Q2)

**Context:** Polarized 11-13h/wk + Z4 intro. Capacity expansion phase.

**Target Rationale:** Aerobic ceiling lift. Z4 intervals intro (5-8 min × 4-6 reps). VO2max measurable +5-10% W12. LBM continue rebuild 65→66.

**Peptide Rationale:**
- MOTS-c continue cycle (re-cycle 6 wk on / 4 wk off)
- NAD+ IV continue
- SS-31 re-cycle (18wk off done dari Q1)
- BPC+TB continue
- **GHK-Cu** ⭐ intro (via KLOW bundle) — connective tissue support

**Cardio Rationale:** Polarized 80/20 — Z2 80%, Z4 20%. Brick 1-2×/wk. Swim continue, bike volume scale.

**Gym Rationale:** 3×/wk compound. Add power output (jump, plyo PRN). RPE 7-8.

**Vitamin Rationale:** Stack stable Q1.

**Transition → Mesocycle 3 (2028Q3 Hybrid Str-End):** Aerobic capacity expansion measurable. Volume scale 13-14.5h/wk. Brick 2×/wk.

## MESOCYCLE 3: HYBRID STRENGTH-ENDURANCE (2028Q3)

**Context:** Lift + endurance integration. Brick 2×/wk. Hybrid athlete profile.

**Target Rationale:** Maintain LBM 66 + improve VO2 + half-marathon pace test. BF stable 10-11%.

**Peptide Rationale:**
- MOTS-c continue
- BPC+TB continue (volume peak)
- **TB-500 emphasis** (tendon stress 13-14.5h volume)
- NAD+ IV continue
- SS-31 off (18wk reset)

**Cardio Rationale:** 13-14.5h/wk hybrid. 2×/wk brick (bike→run). Swim continue 2×/wk. Long bike Sabtu 3-4h.

**Gym Rationale:** 2×/wk compound (reduce frequency, volume preserve). Power + strength integration.

**Vitamin Rationale:** Stack stable. Mg up 500mg PRN (cramping prevention volume tinggi).

**Transition → Mesocycle 4 (2028Q4 Performance Peak):** Hybrid base solid. Volume scale 14.5-16h/wk. Half-marathon TT execution.

## MESOCYCLE 4: PERFORMANCE PEAK (2028Q4)

**Context:** Peak 14.5-16h/wk + half-marathon TT. Aerobic ceiling test, MC3 capstone.

**Target Rationale:** Half-marathon TT < 2:00 (benchmark for MC4 70.3 ready). LBM 66 maintain. BF 10% stable.

**Peptide Rationale:**
- MOTS-c continue (peak mito demand)
- SS-31 re-cycle (5wk on peak phase)
- NAD+ IV continue
- BPC+TB continue
- **ARA-290** ⭐ intro (endurance + anti-inflam dual)

**Cardio Rationale:** Peak 14.5-16h/wk. 2-3×/wk brick. Long sessions: bike 4-5h, run 90-120 min. Half-marathon TT W10-12.

**Gym Rationale:** 2×/wk strength preserve. Reduce accessory.

**Vitamin Rationale:** Stack stable. Electrolyte protocol race-tested pre-TT.

**Transition → MC4 (2029Q1 Race-Build Base 70.3):** Engine ready untuk race-specific build. Half-marathon TT < 2:00. Tendon load tolerance proven.

## SHORT/LONG TERM ALIGNMENT

**Short term:** Aerobic engine maximization, half-marathon TT < 2:00 benchmark.

**Long term:** MC3 = engine foundation untuk MC4 (70.3) dan MC5 (Full IM). Polarized base = lactate clearance superior, RER profile favorable. Mito biogenesis compound effect 12 bulan = race-day fuel efficiency unmatched.

**Risk skip:** Tanpa MC3 polarized build, MC4-MC5 ceiling capped, race performance plateau early.
