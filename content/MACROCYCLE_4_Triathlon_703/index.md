## MESOCYCLE 1: RACE-BUILD BASE 70.3 (2029Q1)

**Context:** Race-specific build start. OWS (open water swim) entry. Race-pace work introduced.

**Target Rationale:** Aerobic base from MC3 maintain. Race-pace skill build: swim 1500m OWS, bike 90km Z2-Z3, run 21km Z2. Half-IM specific load progressive.

**Peptide Rationale:**
- **MOTS-c** continue (mito demand sustained)
- **BPC-157 + TB-500** cycle (tendon load weekly)
- **GHK-Cu (KLOW)** ⭐ continue
- **NAD+ IV** continue (ATP support volume tinggi)

**Cardio Rationale:** Race-build base 12-14h/wk. OWS 1×/wk minimum. Bike long Sabtu 3-4h. Run progressive.

**Gym Rationale:** 2×/wk strength preserve. Single-leg work emphasis.

**Vitamin Rationale:**
- Tier S continue
- Creatine 5g continue
- Iron strict monthly (running volume tinggi)
- Vit C cap 500mg

**Transition → Mesocycle 2 (2029Q2 Peak):** Base solid, no overuse injury. Volume scale peak phase.

## MESOCYCLE 2: RACE-BUILD PEAK 70.3 (2029Q2)

**Context:** Peak fitness + half-IM rehearsal Sabtu. Heat acclim Hyperthermic Conditioning.

**Target Rationale:** Half-IM rehearsal full distance simulation Sabtu Q2-W4 + W8 + W12. Heat acclim 60 min sauna 80-90°C 4×/wk W8-W12.

**Peptide Rationale:**
- Stack Q1 continue
- **TB-500 emphasis** — tendon stress peak load
- **GHK-Cu** continue (heat stress recovery)

**Cardio Rationale:** Peak 14-16h/wk. Brick 2-3×/wk. Half-IM rehearsal monthly.

**Gym Rationale:** 1-2×/wk maintenance.

**Vitamin Rationale:**
- Electrolyte protocol race-tested W4
- Heat acclim → electrolyte demand naik
- Mg 500mg cramping prevention

**Transition → Mesocycle 3 (2029Q3 Race):** Half-IM rehearsal sub-6:00 achieved. Heat acclim complete. Taper Q3 W-3 to W-1.

## MESOCYCLE 3: RACE EXECUTION & RECOVERY (2029Q3)

**Context:** Taper + RACE 70.3 + post-race recovery.

**Target Rationale:** Race target sub-6:00 (stretch sub-5:30). Post-race recovery 4-6 wk: HR variability baseline, sleep baseline, zero residual injury.

**Peptide Rationale:**
- Taper week: drop volume-heavy stack
- **BPC+TB intensive** race week + 2 wk post (recovery)
- **Thymosin α-1** ⭐ intro (immune post-race)
- **MOTS-c** off cycle post-race (mito reset)

**Cardio Rationale:**
- W-3: volume 60%
- W-2: 40%
- W-1: race week 30%
- Race day: 70.3
- Post-race W+1 to W+4: active recovery

**Gym Rationale:** Off pre-race week. Post-race light W+2 onwards.

**Vitamin Rationale:**
- Race-day fuel protocol locked
- Post-race Zinc + Vit C bump (immune)
- Electrolyte race-day protocol executed

**Transition → MC5 (2029Q4 IM Base Start):** 70.3 completed sub-6:00. Recovery 4-8 wk done. IM 140.6 build entry.
