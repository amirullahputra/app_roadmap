## MESOCYCLE 2: HEAVY CUT (2026Q3)

**Context:** Heavy Cut start, F1 phase. BB 78→71kg, BF 28→22% di 13 minggu. Peptide F1 dominant offense, Z2 14h/wk fat ox foundation.

**Target Rationale:** Rate ~0.5 kg/wk = aggressive tapi sustainable dengan stack heavy support. -6% BF di 13 wk = visible transformation milestone, momentum psikologis penting.

**Peptide Rationale (F1 Stack — Score ≥ 25):**
- **Tesamorelin** (F1 76.2): GHRH receptor agonist → physiological GH pulse → visceral lipolysis + IGF-1 +180%. Daily 1mg pagi fasted.
- **Retatrutide** (F1 70.3): GLP-1 triple agonist (GLP-1/GIP/glucagon) → appetite suppression + EE↑. Titrate 2 → 4-6 mg weekly.
- **SS-31** (F1 43.2): Cardiolipin-targeting tetrapeptide → mito ATP efficiency. Cycle 5wk on / 18wk off, daily 5on/2off.
- **Ipamorelin** (F1 36.7): GHRP nocturnal → selective GH pulse SWS sync, LBM preserve. 300mcg pre-sleep.
- **NAD+ IV** (F1 35.0): Sirtuin substrate, ATP + mito ETC. 3×/wk IV slow push 20-30 min.
- **MOTS-c** (F1 30.0): Mito-derived peptide, AMPK activation → fat ox + insulin sensitization. 3×/wk pre-cardio seeding.

Synergy: Tesa + Ipa = Datbtrue GHRH+GHRP amplification. SS-31 + NAD+ + MOTS-c = mito 3-layer (membrane + substrate + biogenesis).

**Cardio Rationale:** Walking incline Z2 14 jam/wk (2h × 7d). Z2 95% polarized base — fat ox optimal, recovery cost rendah, LBM preserve. No HIIT (cortisol cost tidak worth). Foot-strike loss compensated dengan iron EOD.

**Gym Rationale:** 2×/wk compound (Sel lower, Jum upper). RPE 7-8 maintain (no PR push). Squat 1RM hold = LBM proxy. Minimum effective dose Helms ISSN 2014 untuk LBM preserve selama cut heavy.

**Vitamin Rationale (Tier S Mandatory):**
- Vit D3 calibrated (DHCR7+CYP2R1+GC compound deficiency)
- K2 MK-7 150mcg (Ca routing + bone load walking entry)
- Methylfolate 800mcg + Methylcobalamin 1000mcg (MTHFR pair, trap risk)
- EPA/DHA 2500-3000mg rTG (FADS1 TC marine mandatory)
- Iron Bisglycinate 45-60mg EOD + Vit C 250mg (TMPRSS6 A-allele Stoffel protocol)
- Mg Glycinate 300-400mg pre-bed (sleep + cortisol blunt)

Tier A overlay: L-Carnitine 2g pre-cardio (F1 fat ox), Zinc 15mg (sweat loss), K Citrate 400mg (Na sensitive). Cap Vit C ≤500mg (MnSOD AG + Paulsen blunting).

**Transition → Mesocycle 3 (2026Q4 Endgame):** W12 DEXA confirm BF ≤ 22% + LBM ≥ 56kg → continue F1 stack into Q4. Re-test 25(OH)D (adipose mobilization release D3) → likely +1000-2000 IU dose adjustment.
