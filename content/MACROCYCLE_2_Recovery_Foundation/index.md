# MACROCYCLE 2 — RECOVERY & FOUNDATION

**Window:** 2027Q3 – 2027Q4 (~26 minggu)
**Trajectory:** BB 57kg → 63kg | BF 6% → 11% | LBM rebuild + aerobic seeding
**Mesocycle Order:** Recovery Sprint → Aerobic Foundation

---

## OVERVIEW

Macrocycle 2 = bridge antara aesthetic peak (MC1) ke triathlon engine build (MC3). 2 mesocycle: (1) controlled rebound + HPG recovery, (2) aerobic base seeding tanpa beban racing. Peptide shift drastis: HGH+HCG keluar, BPC-157 + Kisspeptin-10 + Tesamorelin masuk untuk recovery + HPG normalization. Cardio downshift 5h/wk (post-peak) → 6-8h aerobic foundation.

| Mesocycle | Quarter | BB Start→End | BF Start→End | Theme |
|---|---|---|---|---|
| Recovery Sprint | 2027Q3 | 57→61 | 6→10% | Reverse diet + HPG recovery + tendon prep |
| Aerobic Foundation | 2027Q4 | 61→63 | 10→11% | Z2 6-8h + skill build swim/bike/run |

---

## SCORING FRAMEWORK

Peptide selection prioritas **Recovery score** (4-axis composite di `_global/PEPTIDE_REFERENCE.md`) bukan offense/endurance. Wolverine Bundle (BPC-157 + TB-500), Kisspeptin-10, Tesamorelin (continue dari MC1 endgame), MOTS-c entry seeding.

Vitamin: baseline Tier S/A continue, **Creatine 5g** essential (LBM rebound), Tongkat Ali drop kecuali T free masih < 400.

## MESOCYCLE 1: RECOVERY SPRINT (2027Q3)

**Context:** Post-peak controlled rebound. Reverse diet 250 kcal/wk increase. HPG axis recovery critical setelah 26 minggu deficit deep. BB 57→61, BF 6→10%.

**Target Rationale:** Rebound INTENTIONAL — bukan bounceback uncontrolled. 4kg BB gain di 13 wk = +0.3 kg/wk reasonable. BF naik 4% acceptable jika LBM rebuild. Goal: T free ≥ 500 ng/dL recovery.

**Peptide Rationale:**
- **HGH + HCG TAPER** dari MC1 → drop end of W4
- **Tesamorelin continue** 1mg daily (visceral lipolysis maintain saat refeed)
- **Kisspeptin-10** ⭐ (HPG upstream, T recovery)
- **BPC-157 + TB-500 (Wolverine Bundle)** ⭐ ENTRY — tendon prep MC3+
- **MOTS-c** continue cycle (mito seeding bridge MC3)
- **Thymosin α-1** continue (immune deficit-recovery)

**Cardio Rationale:** Z2 5-6h/wk easy. Walking dominant, swim PRN. Bukan training pressure — recovery active. Cortisol monitor mandatory pertama 4 minggu.

**Gym Rationale:** 3×/wk compound RPE 6-7. Surplus + creatine + lift = LBM rebuild optimal. Bench/squat/dead progressive load.

**Vitamin Rationale:**
- Tier S continue (genomics-flagged tidak berubah)
- **Creatine 5g daily** ⭐ MANDATORY (LBM rebuild)
- Boron taper, Tongkat Ali continue jika trial positive
- Methylcobalamin 1000mcg (continue post-deficit B12 demand)

**Transition → Mesocycle 2 (2027Q4):** End of W12. T free ≥ 500, LBM ≥ 55kg, no rebound overshoot >12% BF. Stack pivot: drop HCG taper-out, intro 3-discipline skill build.

## MESOCYCLE 2: AEROBIC FOUNDATION (2027Q4)

**Context:** Aerobic engine seeding. 3-discipline skill build: swim 1000m → 1500m, bike 60 → 90 min, run 30 → 60 min. Foundation untuk MC3 polarized build.

**Target Rationale:** BB 61→63 maintain mild surplus. BF stable ~11%. LBM continue rebuild. Aerobic markers: VO2 at VT1 baseline test, lactate threshold pace establish.

**Peptide Rationale:**
- **BPC-157 + TB-500** continue cycle (tendon load adapting to running)
- **MOTS-c** continue (mito biogenesis pre-MC3)
- **Tesamorelin** taper down → drop end Q4
- **Kisspeptin-10** continue jika T free belum stable
- **NAD+ IV** intro (mito ATP support endurance entry)

**Cardio Rationale:** Z2 6-8h/wk. Brick session intro Sabtu (bike → run 30+15 min). Swim 2×/wk (technique focus, drills). Run progressive ramp +10%/wk.

**Gym Rationale:** 3×/wk compound continue. Hybrid prep: add unilateral (single leg) untuk run prep. Volume maintain.

**Vitamin Rationale:**
- Tier S continue
- Creatine 5g continue
- Iron monitor (running entry = foot-strike loss compounding)
- Vit D3 + K2 continue (bone load running)

**Transition → MC3 (2028Q1 Mito Base Build):** Aerobic engine seeded, ready volume scale 8-11 jam/wk. Stack pivot: MOTS-c + SS-31 + NAD+ dominant (mito layer). Tendon prepped via BPC+TB.

## SHORT/LONG TERM ALIGNMENT

**Short term:** Reset metabolic damage, HPG recovery, foundation skill 3-discipline.

**Long term:** MC2 = make-or-break bridge ke Ironman 2030. Skip recovery = MC3 volume tidak naik. HPG masih suppressed → libido drop, training adaptation pelan. Tendon kurang prep → MC3+ injury risk tinggi. **MC2 = invest 6 bulan, save 24 bulan downstream.**
